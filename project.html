Home Decor

Home Decor is an online shopping website for home decoration and accessories.

  • Users can browse/comment all products' details, add them to cart and get their order details.
  • If user has admin authorization, he/she can edit/delete/list a product.
  • If user has admin authorization, he/she can edit/delete/list a casual user.
  • It is implemented in HTML, Bootstrap, React / Redux / Redux-thunk, MERN, JWT.

For more details please visit website & github.
